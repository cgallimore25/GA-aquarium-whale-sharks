Running the 'OV_species_tree.m' reproduces the phylogenetic tree from Figure S1.

The .phy file was generated from: https://www.ncbi.nlm.nih.gov/Taxonomy/CommonTree/wwwcmt.cgi by adding the species list 'OV_species_sci_names.txt' file and exporting as 'phylip tree'.

'OV_species.xlsx' contains the broad class (e.g. Fish, Shark/Ray, Reptile), common name, scientific name, LSID from the World Register of Marine Species (WoRMS; https://www.marinespecies.org/index.php), and the Author, Year of the original species description.
